# Public Assets

This directory contains static assets for the portfolio website.

## Required Files

### resume.pdf
Add your resume PDF file here. The file should be named `resume.pdf` and will be accessible at `/resume.pdf`.

To add your resume:
1. Export your resume as a PDF
2. Name it `resume.pdf`
3. Place it in this `public` directory

### favicon.ico
Add your favicon file here. You can generate one at:
- [Favicon Generator](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

## Optional Files

You can add any other static assets here, such as:
- Images for projects
- Logos
- Other downloadable files

All files in this directory will be accessible at the root URL path (e.g., `/favicon.ico`, `/resume.pdf`).
